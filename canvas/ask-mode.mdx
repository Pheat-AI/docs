Ask Mode allows you to interact with the canvas as if you're having a conversation with ChatGPT. It's a versatile tool for creating prompts, planning your work, or exploring ideas.

## Using Ask Mode

You can think of Ask Mode like having a ChatGPT conversation with the canvas. Use it to create prompts, plan your work, or anything else you can imagine.

## Extracting Prompts from Images & Videos

Ask Mode can be used to extract a prompt from an image. This feature allows you to seamlessly integrate visual elements into your ideation process, or recreate images and videos you see online.
